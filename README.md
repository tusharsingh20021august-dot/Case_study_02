# Share Market 

**Project Overview**
**EDA Project | Python • Pandas • Matplotlib • Seaborn**
This project focuses on analyzing share market data using Python.
The goal is to understand price behavior, identify trends, calculate technical indicators
and generate insights useful for trading or investment decisions.


**This analysis covers**:
Importing & cleaning stock data
Visualizing price movements
Calculating moving averages
Calculating RSI
Calculating MACD
Understanding market trends
Distribution of daily returns
Insights that can help in trading & investment decisions

**Analysis & Visualizations**
Each visualization helps interpret market behavior.
Below are all indicators and their plot explanations:

**Simple Moving Average (SMA)**
What it shows:
SMA is used to smooth price data and highlight market trends.

**Relative Strength Index (RSI)**
What it shows:
RSI indicates whether the stock is overbought or oversold.
RSI > 70 → Overbought
RSI < 30 → Oversold

**Price & Volume Chart**
What it shows:
Shows how trading volume affects price movement.

**MACD (Moving Average Convergence Divergence)**
What it shows:
MACD identifies momentum changes and trend strength.

MACD Line
Signal Line
Histogram (momentum)
MACD crossing above Signal → Buy signal
MACD crossing below Signal → Sell signal
Useful for identifying entry/exit points.

# Conclusion

This analysis provides a clear understanding of stock price trends, momentum, and volatility using basic but powerful technical indicators.
These insights can support trading, investing, or risk analysis decisions.
